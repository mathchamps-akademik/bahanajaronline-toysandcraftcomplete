Panduan Spreadsheet Hasil

Guru tidak perlu setup teknis tambahan. Jawaban siswa otomatis masuk ke dashboard hasil online, lalu bisa difilter dan diekspor menjadi CSV untuk dibuka di Google Sheets atau Excel.

CATATAN GURU
  • Sebelum kelas, pastikan pilihan Cabang dan Nama Teacher di test-config.js sudah sesuai.
  • Saat kelas berlangsung, buka dashboard hasil di tab terpisah dan klik Refresh Data setelah siswa submit.
  • Gunakan Export CSV jika admin membutuhkan arsip di Google Sheets atau Excel.

1. Buka Link Test

Guru membagikan link Pre Test atau Post Test sesuai pertemuan. Siswa mengisi email, nama lengkap, cabang, dan nama teacher.

2. Pastikan Semua Submit

Di awal dan akhir kelas, guru mengecek dashboard hasil. Klik Refresh Data sampai nama siswa muncul.

3. Gunakan Filter

Guru bisa memilih Day, jenis test, nama teacher, cabang, atau mencari nama siswa agar data kelas lebih mudah dibaca.

4. Export Spreadsheet

Klik Export CSV untuk mengunduh hasil yang sedang tampil. File CSV bisa dibuka di Google Sheets atau Excel.

5. Edit Cabang & Teacher

Admin hanya perlu mengubah daftar cabang dan nama teacher di test-config.js. Konfigurasi penyimpanan hasil tidak perlu diubah.
